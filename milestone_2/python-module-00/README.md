# Python Module 00

This project contains exercise modules for Python learning.

## Project Structure

```
python-module-00/
├── ex0/                    # Exercise 0 module
│   ├── __init__.py        # Package initialization
│   └── ft_plot_area.py    # Exercise functions
├── main.py                 # Main entry point for testing
└── README.md              # This file
```

## Running the Project

To test the project, run:

```bash
python main.py
```
